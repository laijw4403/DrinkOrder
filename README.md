# Order Drinks APP
> conveniennce to order drinks for teatime

## Background
The [KEBUKE](http://kebuke.com/) is famous for their black tea, we can use this app to order customized drinks, and calculate the cost.

## Features
- Menu
- Order customized drinks
- Order List
- Modify order
- Delete order
